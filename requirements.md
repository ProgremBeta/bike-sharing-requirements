# Sistema de préstamo de bicicletas
  
## Stakeholders
  * Usuarios
  * Administradores
  * Local de bicicletas
  * Equipo de mantenimiento
  * Sistema de pagos

## Requisitos funcionales
  * El sistema debe permitir al usuario hacer register y login
  * El sistema debe permitir mostrar ls disponibilidad de la bicicletas
  * El sistema debe mostrar una vista previa y permitir una vista detallada de cada bicicleta
  * El sistema debe permitir realizar un prestamos prestamo
  * El sistema debe permitir realizar el pago deacuerdo con el prestamo
  * El sistema debe permitir realizar filtros al usuario
